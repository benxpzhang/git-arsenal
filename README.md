# Git Arsenal

AI-powered GitHub repository search engine with 3D galaxy visualization.

Search 150k+ open-source repositories using natural language, explore connections between projects in an interactive 3D star map, and let an AI agent find what you need via MCP tools.

## Features

- **Semantic Search** — HyDE-enhanced vector search over repository structure trees
- **3D Galaxy** — Force-directed graph visualization of repository clusters with similarity-based particle connections
- **AI Agent Chat** — Conversational interface powered by MCP tools for natural language repo discovery
- **Language Filters** — Toggle visibility by programming language in the galaxy view
- **Cross-cluster Exploration** — Click any node to dynamically load its global neighbors

## Architecture

```
packages/
├── api/          # FastAPI backend — search, galaxy, auth APIs
├── web/          # Next.js frontend — chat, galaxy 3D visualization
└── mcp-server/   # MCP tool server — exposes search to AI agents
```

### Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Next.js 15, React 19, Tailwind CSS, 3d-force-graph |
| Backend | FastAPI, SQLAlchemy, Qdrant (vector DB), PostgreSQL |
| AI | OpenAI-compatible embeddings, GLM for HyDE |
| Agent | MCP (Model Context Protocol) |

## Prerequisites

- Python 3.11+
- Node.js 18+
- Docker & Docker Compose (for Qdrant + PostgreSQL)

## Setup

### 1. Clone & install dependencies

```bash
git clone https://github.com/<your-username>/git-arsenal.git
cd git-arsenal

# Backend
cd packages/api
pip install -r requirements.txt

# Frontend
cd ../web
npm install

# MCP server
cd ../mcp-server
npm install
```

### 2. Start infrastructure

```bash
# From project root
docker compose up -d   # Starts Qdrant + PostgreSQL
```

### 3. Configure environment

```bash
# Root-level backend config
cp .env.example .env
# Edit .env — fill in LLM_API_KEY and EMBED_API_KEY

# Frontend config (optional, for AI agent mode)
cp packages/web/.env.local.example packages/web/.env.local
```

### 4. Prepare data

The galaxy visualization requires pre-computed data files in `packages/api/data/`:

| File | Description |
|------|-------------|
| `repos_meta.jsonl` | Repository metadata (one JSON object per line) |
| `galaxy_edges.npz` | Similarity edges between repos (`src`, `dst`, `sim` arrays) |
| `positions_3d.npy` | 3D coordinates for each repo |
| `repo_leaf_labels.npy` | Cluster leaf label for each repo |
| `cluster_tree.json` | Hierarchical cluster tree structure |

These files are generated by the data pipeline (see `scripts/ingest.py`). Place or symlink them into `packages/api/data/`.

### 5. Initialize database & ingest data

```bash
python scripts/init_db.py
python scripts/ingest.py
```

### 6. Run

```bash
# Terminal 1 — Backend API
cd packages/api
uvicorn server:app --host 0.0.0.0 --port 8003

# Terminal 2 — Frontend
cd packages/web
npm run dev
```

Open http://localhost:3000 in your browser.

## MCP Integration

Git Arsenal exposes an MCP server for AI agents (Cursor, Claude Desktop, etc.):

```json
{
  "mcpServers": {
    "git-arsenal": {
      "type": "stdio",
      "command": "node",
      "args": ["packages/mcp-server/index.js"],
      "env": {
        "ARSENAL_API_URL": "http://localhost:8003"
      }
    }
  }
}
```
